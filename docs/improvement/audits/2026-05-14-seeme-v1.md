## Gaps (numbered)

1. **`src/lint/alignment.ts`** — no visualization of which `│` is offset, just a line-number error. Future-me hits a failure with low context.
2. **No CI integration test against real Anthropic API** — caching path is unverified end-to-end (passes only against mocks + Ollama).
3. **`src/providers/index.ts createOllama`** — rebuilt per call. Fine for CLI; wasteful in library mode (MCP).
4. **`src/cli.ts formatUsage`** — when provider returns zero usage, line correctly suppresses, but no test asserts this. Could regress.
5. **`OLLAMA_HOST` zod chain in `src/env.ts`** — convoluted; readable refactor possible.
6. **`src/prompt/system.ts`** — SKILL.md path sync-loaded at import. Unfriendly crash if `ascii-flow-diagrams/SKILL.md` moves.
7. **No `package.json` `lint` script + ESLint config** — `tsc --noEmit` is the only static check.
8. **No `seeme history` cross-session aggregation** — JSONL log exists but no analysis.

## Elevations (lettered)

A. **Public npm publish for `@diagrammer/cli`** — `npx seeme "..."` for anyone, no clone needed.
B. **Cornhole Arena diagram integration** — connect SEEME to red-e-play's broader project surface.
C. **`seeme stats` cross-session aggregation + dashboard** — JSONL already there; needs viz.
D. **Sub-classifier improvements** — more no-diagram failure types (currently 4; could be 8+: truncated-without-fence, json-only, …).
E. **Web preview app (Next.js)** — was Phase 2 in the original plan; replaced by MCP server but the web app remains latent.
F. **Anthropic 1h cache (`SEEME_LONG_CACHE=1`)** — wiring in place but unverified end-to-end.
