# You-Sir Juan Upstream Repo Ledger

## Purpose

This is the canonical copied upstream-repository ledger inside the AI Skills Library.

Master platform planning lives in:

```text

```

This repo keeps the portable skill-library copy so agents can quickly discover which upstream tools, repos, and skill systems belong to the You-Sir Juan ecosystem.

# Usage Rule For AI Agents

Use this document when deciding:

- what repo/tool belongs to which skill family
- what the tool does
- how it supports You-Sir Juan
- whether it is active, planned, candidate, or reference
- which platform domain should own the tool

# Maintenance Rule

When a new upstream repo is discussed, add it here first.

Then, if it becomes important enough, also add it to:

- the relevant skill-family guide
- `docs/yousirjuan-platform-skills-master.md`
- the main README navigation if it becomes a top-level family
- a future individual `SKILL.md` file if it should become an executable/reusable skill
