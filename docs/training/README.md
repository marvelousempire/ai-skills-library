# Training system

Hands-on drills that turn the patterns into reflex. Each drill is verifiable and self-contained.

## The 8 drills

| # | Drill | Time | What you practice |
|---|---|---|---|
| d01 | [`d01-skill-scaffold.md`](drills/d01-skill-scaffold.md) | 15 min | Scaffold a new skill from the template |
| d02 | [`d02-add-container-labels.md`](drills/d02-add-container-labels.md) | 10 min | Apply the label schema to a compose file |
| d03 | [`d03-cross-reference-ripple.md`](drills/d03-cross-reference-ripple.md) | 10 min | Add a skill + ripple all the references |
| d04 | [`d04-doctor-script.md`](drills/d04-doctor-script.md) | 20 min | Write a doctor script from the template |
| d05 | [`d05-makefile-make-ui.md`](drills/d05-makefile-make-ui.md) | 20 min | Build a `make ui` one-command Makefile |
| d06 | [`d06-plan-first.md`](drills/d06-plan-first.md) | 15 min | Given a task, write the plan |
| d07 | [`d07-gap-audit.md`](drills/d07-gap-audit.md) | 15 min | After a fake ship, run the audit |
| d08 | [`d08-decision-record.md`](drills/d08-decision-record.md) | 15 min | Pick A over B + write the record |

## How drills work

Each drill has:
- **Starting state** — what files / state exist before
- **Target state** — what should exist after
- **Verification** — exact command + expected output
- **Solution** — link to a real example in the repo when applicable

## Why

Reading docs teaches the pattern. Drills make it reflex.
