# 13 — Reasoning, Evaluation & Red-Team Skills Family

## Purpose

This family covers advanced reasoning, critique, premortem analysis, explainability, adversarial evaluation, security testing references, AI red-team awareness, and defensive validation workflows.

Use this family when the task involves:

- agent reasoning
- critique
- steelmanning
- premortem review
- explainability
- AI evaluation
- safety checks
- security validation
- defensive red-team planning
- adversarial testing frameworks
- risk discovery before launch

---

# Safety Boundary

This family is for defensive, educational, and validation work only.

Do not use these resources to plan, enable, or execute unauthorized intrusion, abuse, malware, credential theft, evasion, or harmful cyber activity.

Allowed use:

- defensive security testing
- internal controls validation
- AI safety evaluation
- product risk review
- detection engineering
- tabletop exercises
- responsible red-team planning
- compliance-oriented testing
