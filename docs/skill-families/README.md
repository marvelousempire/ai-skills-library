# Skill Families Index

## Purpose

This folder separates the AI Skills Library into clear skill families so humans and AI agents can quickly route a task to the correct skill group.

Use this folder as a lightweight routing map before opening deeper skill catalogs.

# AI Routing Rule

When unsure:

1. Identify the user's goal.
2. Pick the closest skill family.
3. Open the family file.
4. Follow the linked catalog or skill path.
5. Prefer project-specific skills over generic skills when available.

---

# Maintenance Rule

Every new skill family should include:

- purpose
- use cases
- inputs
- outputs
- related files
- platform role
- AI routing notes
