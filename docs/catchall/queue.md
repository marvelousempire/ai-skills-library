# Catchall Queue

Unmatched requests that no existing skill, rule, agent, or context could handle.
Processed automatically by GitHub Actions on every push and hourly.

Add new items with:
```
## [YYYY-MM-DD] [Short title]
**Received:** ISO-8601 timestamp
**Request:** The exact or paraphrased request
**Session:** Context / project / conversation name
**Attempted matches:** Skill or rule names tried
**Status:** pending
```
