# Orientation Handbook
**managed by the duty and order of AVERY GOODMAN**

> This is the employee handbook for the AI Skills Library. Every AI — whether Claude, GPT, Gemini, or any agent — reads this before operating in the library. Every human contributor reads this before adding to the library. It is organized like a legal code: numbered chapters, numbered sections, clear authority.

## How this handbook is organized

Like a legal code. Like a constitution. Numbered chapters with numbered sections. The numbering is permanent — sections are never renumbered, only added to.

```
orientation/
├── README.md                          ← You are here. The master index.
├── chapter-01-first-principles/       ← What this library is and why it exists
├── chapter-02-the-rules/              ← The governing standards (always-on)
├── chapter-03-procedures/             ← How to create, modify, and file products
├── chapter-04-agent-hierarchy/        ← Who does what and the chain of command
└── chapter-05-growing-the-library/    ← How the library learns and expands
```
