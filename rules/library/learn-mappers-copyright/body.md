# Learn Mappers LLC — app copyright & attribution

When adding or editing **copyright lines**, **legal footers**, **About / credits**, **App Store copyright / seller text**, **Terms / Privacy headers**, or similar **IP attribution** for apps this org ships:

## Required wording

- **Legal entity:** `Learn Mappers LLC DBA AVERY GOODMAN` — **always** render **AVERY GOODMAN** in **ALL CAPS** (never "Avery Goodman" or mixed case unless a court filing requires legal-name matching exactly as filed).
- **Copyright:** use **© 2026** (or the current ship year if the user explicitly updates the rule) with **Learn Mappers LLC DBA AVERY GOODMAN** as the rights holder unless the user gives a different year for a specific asset.
- **IP:** these apps are asserted as **intellectual property** of that entity.
- **UCC:** include the user's reservation language where they ask for it in legal/copyright UI: reference **UCC 1-308** as they use for their products (e.g. alongside copyright in the same block only when the user requests that clause in that surface).

## Scope

- Applies to **all apps** and surfaces the user treats as Learn Mappers LLC product (web, iOS, admin, marketing, status pages, etc.) when you are writing **attribution or legal identity** copy — not a substitute for lawyer-reviewed contracts; follow user edits if they narrow wording for a given document.

## Examples (when the user has requested the full block)

```
© 2026 Learn Mappers LLC DBA AVERY GOODMAN · All rights reserved · UCC 1-308
```

```
This app is the intellectual property of Learn Mappers LLC DBA AVERY GOODMAN.
© 2026 · All rights reserved · UCC 1-308.
```

## Do not

- Invent a different DBA spelling or drop **DBA AVERY GOODMAN** without user direction.
- Lowercase **AVERY GOODMAN** in user-facing attribution strings.
- Add this attribution to surfaces where the user hasn't asked for it (only when writing legal/IP/credits/copyright copy).
