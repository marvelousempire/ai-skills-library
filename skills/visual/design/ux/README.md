# visual / design / ux

**What belongs here:** User experience — mood and method. How it *feels* to be inside the product. The emotional arc. The flow. The psychology. The behavioral design.

UX is NOT what it looks like (that's UI / paint). UX is what it's *like* — the sequence of moments, the friction or ease, the emotional tone throughout the journey.

Currently houses: `ui-ux-pro-max` (covers both UX methodology and UI reference — lives here because the methodology / method side dominates).
