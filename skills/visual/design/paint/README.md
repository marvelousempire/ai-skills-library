# visual / design / paint

**What belongs here:** Pure decoration — color, branding, surface-level visual identity. The things that make something *recognizable* and *beautiful* without changing its structure or behavior. Brand marks, color palettes, texture, decoration.

Paint is the outermost layer. Structure can stay the same; paint changes how it presents. If you repaint a room, the walls are still there.
