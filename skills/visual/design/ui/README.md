# visual / design / ui

**What belongs here:** User interface — the cosmetic visual layer. What it looks like. Buttons, spacing, color, typography, layout, component appearance. The things you can change without changing how the product *behaves* or *feels*.

UI is the surface. It's cosmetic. Changing it makes things look different, not work differently.

Currently empty — ready to receive skills about specific UI patterns, component libraries, or visual system standards.
