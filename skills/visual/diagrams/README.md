The **style spec** defines the format (Unicode box-and-arrow inside fenced `text` blocks, two variants + merged hybrid, eight worked examples). **SEEME** is the CLI that reads the spec as its system prompt and turns any input into a diagram in that style — Ollama-default, with OpenAI / Anthropic / Gemini / Perplexity as bring-your-own-key fallbacks.
