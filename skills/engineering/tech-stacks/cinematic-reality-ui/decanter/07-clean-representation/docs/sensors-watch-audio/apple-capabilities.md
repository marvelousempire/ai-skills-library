# Apple Capabilities

Explainer ID: FSEXP-013  
Source lines: 111-119  
Related IDs: PRD-010-PRD-014; FSTECH-022-FSTECH-026

## Core Point

The PRD calls out several Apple platform capabilities that can make the product feel richer than a standard web demo.

## Capability Map

| Capability | Use |
|---|---|
| TrueDepth / LiDAR | Scan real objects, faces, or environments for scene or export workflows. |
| ARKit / RealityKit world tracking | Provide accurate native motion and spatial context. |
| Core Haptics | Create custom high-quality vibration patterns that feel like liquid movement. |
| Metal Performance Shaders | Accelerate GPU-heavy simulation and post-processing. |
| Spatial Audio | Make pouring and sloshing sound react to device orientation and scene placement. |
| AVFoundation | Record and play high-quality audio. |
| Dynamic Island / Live Activities | Potential later surface for lightweight fluid status or animation. |

## Build Order

These are leverage points, not Slice 1 blockers. Add them after the web prototype proves the core milk-pour illusion.
